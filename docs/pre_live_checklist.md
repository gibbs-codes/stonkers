# Pre-Live Checklist

- [ ] 60+ days profitable paper trading
- [ ] Max drawdown < 15% in paper trading
- [ ] All risk limits tested (daily loss, position limits)
- [ ] Emergency stop loss triggers verified
- [ ] API rate limits understood
- [ ] Funding amount decided ($500-1000 recommended)
- [ ] Win rate > 55% on at least 2 strategies
- [ ] Sharpe ratio > 1.0 on portfolio
- [ ] Correlation between strategies < 0.7
- [ ] Slippage/commission assumptions validated
